************************************ 
QUICK START
************************************

- navigate to the "Tools > Camera Composition" menu to open
  the editor window in Unity
- a "Camera Composition" game object will be added to the hierarchy
- optionally dock the "Camera Composition" editor window into an existing
  layout by dragging and dropping it into place
- turn on/off the grid overlay, and optionally change the grid color for
  better in-scene visibility
- use the golden spiral rotation buttons to adjust the overlay orientation
- adjust target camera position and rotation while monitoring the "Game" camera view
